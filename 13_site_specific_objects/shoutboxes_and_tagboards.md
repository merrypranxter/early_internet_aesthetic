# Shoutboxes and Tagboards

## Definition
A site-specific object family common to old-web, fandom, browser-game, and profile cultures.

## Visual Traits
- small modular graphics
- high symbolic charge relative to size
- often used as declarations of taste, belonging, status, care, or exchange
- repeatable, collectible, and displayable

## Prompt Language
micro-chat widgets, shoutbox columns, tiny public notes, pastel spam, site-owner replies, embedded guest presence

## Pairings
- profile identity systems
- layout grammar docs
- browser world systems
- platform expansions

## Negatives
- oversized luxury-product hero shots
- sleek app icons without context
- generic modern e-commerce widgets
