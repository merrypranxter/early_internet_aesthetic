# Phase 2 Expansion Notes

This expansion deepens **internet_folk_style** into platform-specific subcultures, stronger style merges,
and heavier vocabulary hoards. Phase 1 established the broad architecture. Phase 2 adds the weird little rooms
where the ghosts actually live.

## New emphasis areas
- platform-specific aesthetics: Tumblr, LiveJournal, DeviantArt, Gaia, Neopets, Newgrounds, AIM/MSN, YouTube Poop, forums, imageboards, and desktop-personalization culture
- stronger social-emotional distinctions: overshare melancholy, shrine-like fandom devotion, ironic cruelty, avatar intimacy, anonymous swarm humor
- more prompt-ready merge logic for scene, kawaii, cyber bubblegum, op-art, print damage, and analog archive styles
- expanded reference lists for objects, UI scraps, texture words, action verbs, and emotional tones

## Recommended use
Start from a platform file in `11_platform_expansions`, pair it with one branch file in `02_core_branches`,
then pull a palette from `07_palettes` and a prompt scaffold from `08_prompt_modules`.
