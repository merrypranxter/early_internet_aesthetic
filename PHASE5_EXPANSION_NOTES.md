# PHASE 5 EXPANSION NOTES

Phase 5 extends **internet_folk_style** beyond surface aesthetics into **systems, structures, and user-performance machinery**.

This phase emphasizes:
- theme packs and pseudo-code aesthetics
- layout grammar and page architecture
- profile formats and role/rank identity systems
- browser-game, dollmaker, pet-site, and virtual-world vernacular
- object clusters that behave like social interfaces rather than static props

## Core Shift
Earlier phases focused on vibe, object vocab, platform-specific surface language, and prompt-seed density.

Phase 5 adds the **bones underneath the glitter**:
- how profile pages are arranged
- how users display identity through boxes, badges, top-friends, signatures, ranks, trophies, pets, and inventories
- how early-web code residue acts as visual ornament
- how browser-game systems create icon economies and collectible micro-worlds

## New Folder Summary
- `14_layout_grammar/` = structural templates for pages, dashboards, sidebars, blogs, sig layouts, and clutter choreography
- `15_theme_and_code_packs/` = pseudo-code visual packs, CSS residue aesthetics, HTML table/page logic, Flash/skin/theme systems
- `16_profile_and_identity_systems/` = profile templates, rank ladders, user status systems, OC sheets, fandom profiles, diary/livejournal modes
- `17_browser_games_virtual_worlds/` = pet sites, dollmakers, browser worlds, inventories, pixel economies, forum RPG and avatar-social hybrids

## Phase 5 Intent
This phase is designed to make the repo more useful for:
- AI image prompt generation
- interface and UI scene generation
- synthetic internet archaeology
- fake social-platform worldbuilding
- design references for themed apps, sites, dashboards, archives, and aesthetic engines
