# STYLE_OVERVIEW

## Definition

`internet_folk_style` is a style system for the handmade, damaged, glittered, reposted, collectively-authored visual culture of the web.

It spans:
- old-web homepage chaos
- Y2K chrome optimism
- MySpace self-decoration
- meme vernacular
- screenshot culture
- glitch/data rot
- dead forum archaeology
- mallsoft / vapor-corporate afterlife overlap
- anonymous digital folk art

## Core identity

This style is not one clean timeline. It is a stack of interlocking image habits:

1. **Homemade Web Chaos**  
   Table-layout pages, under-construction GIFs, hit counters, midis, blink tags, guestbooks.

2. **Chrome-Plastic Y2K Optimism**  
   Glass gradients, Frutiger Aero softness, media-player skins, Flash polish, cute synthetic futurism.

3. **Profile Page Selfhood**  
   MySpace customization, glitter names, profile songs, scene glamour, webcam sincerity, social clutter.

4. **Vernacular Meme Logic**  
   Impact captions, recycled templates, MS Paint rawness, screenshot reposting, image-text argument structures.

5. **Digital Damage & Entropy**  
   JPEG rot, scanlines, macroblocks, RGB split, UI corruption, datamosh smear, watermark fossils.

6. **Digital Folk Collectivity**  
   Anonymous authorship, thread sediment, repost mutation, archive ghosts, communal image-making.

## What kills the vibe

- tasteful editorial minimalism
- pristine corporate mockup aesthetics
- cinematic realism with no interface matter
- clean modern flat-design sterility
- over-corrected typography
- luxury-polished brand safety
- anti-clutter instincts
