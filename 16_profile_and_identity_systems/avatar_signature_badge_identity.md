# Avatar Signature Badge Identity

## Definition
A user-identity system where profile structure, metadata, icons, and decorative modules signal personality, status, attachment, obsession, and belonging.

## Core Vibe
Identity performed through boxes, badges, metadata, and visible traces of taste.

## Visual Traits
- tiny symbolic objects carry oversized emotional weight
- metadata becomes autobiography
- profile furniture doubles as social ritual
- public and private tones smear together

## Prompt Language
avatar as mask, sig as manifesto, badges as social proof, icons as tribal jewelry

## Pairings
- site-specific objects
- layout grammar
- platform expansions
- theme/code packs

## Negatives
- blank universal account pages
- default social-media neutrality
- corporate profile cards
- stripped-down biography-only identity

## Prompt Seed Bank
1. A haunted profile page built around this identity system.
2. A shrine-like account layout where every tiny icon signals belonging.
3. A synthetic archive screenshot emphasizing metadata as personality.
4. A maximalist profile skin with status residue and collectible traces.
5. A public-facing self-portrait composed from profile furniture and badges.
