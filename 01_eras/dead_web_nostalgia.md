# DEAD_WEB_NOSTALGIA

## Core identity
The web after abandonment: broken links, expired embeds, archived screenshots, deleted usernames, dead forums, login walls, lost hosting, and fossilized community trace.

## Visual traits
- gray-blue monitor light
- 404 icons
- missing avatars
- screenshot seams
- old timestamps
- static page captures
- locked-thread grave markers

## Emotional tone
haunted, communal, elegiac, absurdly tender, stale but beloved

## Prompt language
archive mausoleum, dead forum necropolis, screenshot archaeology, deleted user epitaph, locked thread tombstone, 404 vigil, old page amber
