# OLD_WEB_90S

## Core identity
Homemade HTML page energy, crude layout logic, visitor counters, under-construction GIFs, guestbooks, webrings, frame sets, blinking tags, autoplay MIDI, bright ugly enthusiasm.

## Visual traits
- table-layout scaffolding
- centered text blocks
- noisy primary colors
- tiny tiles and repeating backgrounds
- weird spacing
- badges and buttons
- page furniture everywhere

## Typical objects
- under construction GIF
- guestbook
- hit counter
- “best viewed in Netscape”
- webring badge
- animated divider bars
- burning text gif
- link-button strips

## Emotional tone
earnest, tacky, charming, obsessive, homemade, overstimulated

## Prompt language
geocities graveyard, angelfire cathedral, blink tag eternity, marquee nightmare, frameset purgatory, table-layout hell, midi autoplay, counter cult, visitor badge

## Negatives
- sleek webapp design
- minimal responsive layout
- tasteful curation
- modern UI cleanliness
