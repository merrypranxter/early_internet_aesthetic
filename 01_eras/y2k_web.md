# Y2K_WEB

## Core identity
Glossy synthetic futurism, chrome-plastic surfaces, smooth gradients, translucent blue-green optimism, media-player skin logic, Flash polish, Windows XP softness, Frutiger Aero overlap.

## Visual traits
- glassy panels
- shiny icons
- chrome reflections
- rounded interface shapes
- cloudy sky and water motifs
- bright synthetic calm
- polished but not adult-serious

## Typical objects
- XP wallpaper references
- Winamp skins
- WordArt
- AIM/MSN emoticons
- Flash intros
- desktop widgets
- frosted buttons
- early digital cameras / webcam softness

## Emotional tone
hopeful, synthetic, bubbly, mall-electronic, cute, overly smooth

## Prompt language
frutiger aero, xp bliss wallpaper, wordart default, msn aim emoticons, blingee glitter, winamp skin religion, flash animation limbo, newgrounds relic, skymall design

## Negatives
- grunge dirt
- harsh realism
- contemporary flat UI
- muted palettes
