# PHASE 6 EXPANSION NOTES

Phase 6 extends the repo from style theory and object libraries into practical prompt-engine scaffolding for historically specific internet-layout imagery.

## Main Additions
- reusable template packs for fake HTML, CSS, BBCode, status-box, profile-card, dashboard, and pet-site structures
- screenshot atlas documents that describe different captured-interface image types as prompt-ready scenes
- a full object taxonomy layer that groups the internet-object ecosystem by function, mood, damage, and era
- extra prompt modules focused on code residue, interface capture scenes, and templated social identity displays

## Why This Phase Matters
Earlier phases described the look and emotional logic of internet aesthetics. This phase adds formal chassis: page anatomy, markup residue, screenshot behavior, and object families that let models stage convincing internet-native compositions instead of generic retro-computer wallpaper.

## Recommended Use
1. choose an era or platform branch
2. choose one layout grammar file
3. choose one template pack
4. choose one screenshot atlas mode
5. garnish with object families, palette, and damage profile
6. optionally merge with another style family from 09_style_merges
