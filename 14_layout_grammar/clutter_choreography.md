# Clutter Choreography

## Definition
How maximalist internet pages remain legible through chunking, repetition, micro-borders, and icon rhythm.

## Core Vibe
Structured clutter, visible social architecture, human-made web logic, identity performed through boxes and repetition.

## Visual Traits
- visible module boundaries
- ornamental separators and repeated metadata
- icon stacks and badge ladders
- page regions with distinct jobs
- hard-edged but expressive layout bones

## Layout Grammar
- content is segmented rather than seamless
- modules become personality fragments
- repetition produces rhythm and familiarity
- metadata is ornamental, not merely functional

## Prompt Language
busy but grouped, repeated modules, visual anchors, icon clusters, separators, mirrored clutter balance

## Pairings
- old web relics
- MySpace customization
- forum signature culture
- browser game dashboards
- digital folk clutter

## Negatives
empty minimalism, random ungrouped noise, Swiss restraint

## Hero Prompt
A maximalist interface relic built around this logic, preserving visible layout bones, decorative metadata, and internet-social residue without collapsing into modern app minimalism.
