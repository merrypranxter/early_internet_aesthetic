# TEXT_TREATMENTS

## Core text modes
- Impact macro text = shouting / blunt meme rhetoric
- Comic Sans = unseriousness, irony, or deliberate cringe
- Arial / system fonts = plain utility or fake neutrality
- WordArt = Y2K performative decoration
- blinking marquee or HTML headline = old-web chaos
- tiny glitter text = profile page ornament
- overwriting and crossout = palimpsest logic

## Useful moves
- caption-image contradiction
- layered labels
- floating tags
- faux academic annotations
- username stacks
- timestamp disagreement
- subtitle-like mood text
