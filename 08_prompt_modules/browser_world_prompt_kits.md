# Browser World Prompt Kits

## Core Idea
Generate browser-native fantasy ecosystems that feel collectible, social, low-stakes, and icon-rich.

## Templates
1. A browser-game dashboard with [pet/economy/room/guild] systems visible, crowded with [icons], [currencies], [inventory objects], and [status widgets].
2. A fake nostalgic pet-site relic using [palette], [pet objects], [shop modules], [trophies], and [microeconomy cues].
3. A virtual-world profile-room hybrid scene packed with [decor], [avatar styling], [collectibles], [friend traces], and [room-builder furniture].
4. A cute obsessive interface where [collection mechanics] are as visually important as the character or pet.
5. A synthetic screenshot from a lost browser world, emphasizing [tiny economies], [forum-social crossover], and [icon-dense reward systems].
