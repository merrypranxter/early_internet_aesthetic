# Object Taxonomy Prompt Kits

## Use Case
This module provides prompt kits for scenes built around compositions driven by object families rather than scenes, letting badges, buttons, widgets, and counters become the subject.

## Build Formula
- choose one era or platform branch
- choose one layout grammar file
- choose one screenshot atlas mode or template pack
- choose one palette and one damage profile
- choose one object family or subject archetype
- decide whether the output is a screenshot, shrine page, collage, portrait, dashboard, or desktop capture

## Prompt Additives
legacy UI residue, screenshot crop awkwardness, tiled backgrounds, code fossils, repost wear, visible personalization labor, internet clutter as composition engine

## Example Seeds
1. a legacy web scene built from this kit where the interface furniture is as important as the subject
2. a portrait or object study wrapped in code residue and screenshot capture logic
3. a hyper-detailed fake screenshot using these modules to sell historical platform specificity
4. an abandoned or haunted version of the same interface with damage, dead links, and archive patina
5. a maximalist collage where these modules become decorative syntax instead of pure function
