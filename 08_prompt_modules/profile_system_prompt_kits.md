# Profile System Prompt Kits

## Identity Through Interface
Profiles are not neutral containers. They are stage sets, confessionals, trophy cabinets, and social spellbooks.

## Negative Anchors
Avoid generic influencer dashboards, blank account pages, flat corporate identity cards, and oversimplified “profile picture only” compositions.
