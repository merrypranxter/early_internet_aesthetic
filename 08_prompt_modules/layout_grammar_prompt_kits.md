# Layout Grammar Prompt Kits

## Purpose
Prompt packs for building images, interface scenes, and synthetic screenshots that feel structurally native to old-web, MySpace, forum, blog, and browser-game design.

## Prompt Templates
1. A [platform] page composed with [layout family], visible [modules], [objects], and [metadata], preserving [emotional tone] and resisting modern product minimalism.
2. A synthetic screenshot of a [profile/dashboard/forum/blog] where [identity system] is expressed through [badges, sidebars, signatures, widgets], with [damage/texture] and [palette].
3. A maximalist digital relic using [table logic / box stacks / feed flow / sig-strip architecture], layered with [site-specific objects] and [platform-specific residue].
4. A fake social artifact where the layout itself tells the story: [public comments / top friends / rank bars / inventory grids / mood icons] dominate the composition.
5. An internet-folk shrine built from [page architecture] and [object family], with [glitter / CRT bleed / jpeg residue / browser chrome] acting as material texture.

## Useful Grammar Slots
- layout family
- object density
- metadata visibility
- social traces
- customization aggression
- damage level
- emotional mode
- platform lineage
