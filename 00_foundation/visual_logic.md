# VISUAL_LOGIC

## Repeating structures
- windows inside windows
- comments layered over images
- icon swarms
- stickers on top of UI
- browser frames treated as architecture
- repeated captions or tags
- profile widgets as ornamental blocks

## Density
Lean dense.
Sparse versions can work, but the native logic is usually additive and accumulative.

## Framing
Useful frame devices:
- browser windows
- CRT/monitor edges
- screenshot borders
- chat frames
- gallery grids
- profile modules
- popup stacks

## Scale behavior
The aesthetic tolerates weird scale shifts:
- tiny text + massive image
- giant icons
- floating buttons
- oversized emoticons
- absurd badge piles

## Contrast
Good contrast pairs:
- glossy gradients + crusty JPEGs
- sacred layout + stupid meme object
- bright candy palette + archive sadness
- corporate clean geometry + repost grime
