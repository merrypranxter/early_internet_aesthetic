# CORE_PRINCIPLES

## 1. The web is not clean
Useful internet aesthetics preserve:
- compression
- crop marks
- browser chrome
- tool marks
- UI interruptions
- visual leftovers

## 2. Reuse is part of the beauty
The copy of the copy of the copy often looks better than the “original” because it carries social history.

## 3. Clutter communicates
Window stacks, badges, stickers, glitter, captions, counters, comments, and overlay junk are not noise to be removed. They are evidence.

## 4. Irony and sincerity coexist
A profile page can be obnoxious and heartfelt at the same time.
A meme can be stupid and rhetorically sharp at the same time.

## 5. Interface is part of the image
Tabs, taskbars, windows, comment boxes, cursor trails, buttons, input fields, and error dialogs can function as architecture, framing, or ornament.

## 6. Damage is information
JPEG rot, scanlines, and datamosh melt are not just texture. They indicate time, platform, transit, and lineage.

## 7. Collective authorship matters
Many internet images feel authored by a crowd, not by one sacred designer. That collective weirdness should stay intact.

## 8. Over-customization is a feature
MySpace logic matters because selfhood is performed through decoration, not minimalist restraint.
