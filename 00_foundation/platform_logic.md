# Platform Logic

A platform is not just a host. It is a visual machine that teaches users how to decorate, crop, type, react,
and perform identity. Early internet aesthetics become most interesting when the platform's social rules are treated
as visual grammar.

## Core claim
Different platforms produce different image behaviors:
- dashboards create collage and quote-fragment aesthetics
- journals create confession and timestamp intimacy
- forums create banners, signatures, and rank badges
- avatar economies create ornamental layering and item-flex logic
- anonymous boards create stripped-down low-fi swarm authorship
- Flash portals create thick-outline cartoon aggression and loading-screen bravado

## Prompt implication
When prompting, do not just ask for a look. Ask for a behavior:
- reblogged instead of authored
- customized instead of designed
- reposted instead of published
- archived instead of finished
