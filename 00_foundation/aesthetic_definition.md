# AESTHETIC_DEFINITION

## Definition

Internet folk style is the aesthetic language of the web as it is actually lived: handmade, reposted, cropped, compressed, personalized, broken, communal, and semi-anonymous.

It is not identical to:
- retro computing
- tech nostalgia
- vaporwave
- cyberpunk
- corporate Y2K branding
- generic “glitch art”

It overlaps with all of those, but its center is **vernacular online image culture**.

## Keywords

homemade web chaos, screenshot culture, digital folk art, browser residue, meme rhetoric, profile customization, anonymous authorship, repost mutation, archive ghosts, interface relics

## Useful shorthand

- old-web relic
- screenshot quilt
- profile-page shrine
- thread sediment
- repost palimpsest
- glitch-folk collage
- browser archaeology

## What matters most

1. visible social trace  
2. visible mediation by interface  
3. some degree of damage, clutter, or residue  
4. emotional contradiction: earnestness + irony  
5. modular remixability
