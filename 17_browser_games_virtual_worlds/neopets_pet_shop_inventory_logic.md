# Neopets Pet Shop Inventory Logic

## Definition
A browser-native micro-world where icons, currencies, collections, pets, items, and social status structures create a playable interface mythology.

## Core Vibe
Cute optimization, collectible obsession, and tiny economies pretending to be worlds.

## Visual Traits
- icon-dense dashboards
- inventories and currencies as visual anchors
- collectible objects imply history and care
- interfaces feel game-like even when static

## Prompt Language
pet panels, neopoints-style currencies, inventory icons, shopkeeper booths, trophy shelves, plushies, paintbrush economies

## Pairings
- dashboard tile systems
- profile identity docs
- site-specific object docs
- kawaii / cyber-bubblegum style merges

## Negatives
- realistic AAA game UI
- sparse neutral dashboards
- generic finance-app icon grids
- modern app-store cleanliness

## Prompt Seeds
1. A browser-game dashboard shrine built around this system.
2. A synthetic archival screenshot of a beloved collectible economy.
3. A room/inventory/profile hybrid page overloaded with tiny rewards.
4. A fake pet-site relic with emotional attachment encoded in interface modules.
5. A nostalgic economy screen full of charming low-stakes obsession.
