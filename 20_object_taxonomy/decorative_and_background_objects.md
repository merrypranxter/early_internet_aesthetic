# Decorative And Background Objects

## Taxonomy Scope
This file groups decorative and background objects within the broader internet-folk style system.

## Main Families
- blinkies, background tiles, glitter text, page dolls, custom cursors, dividers, headers, floating PNG stickers
- each family can appear as literal UI furniture, floating collage matter, iconographic relic, or symbolic shorthand

## Era Range
- old web relics
- Y2K and MySpace customization culture
- early social-web residue
- forum and gallery communities
- meme and repost archaeology

## Aesthetic Roles
- functional object: something the page or platform actually uses
- decorative object: something added for personality or clutter
- identity object: something that signals taste, mood, status, or affiliation
- relic object: something that shows age, damage, dead links, or prior circulation

## Prompt Language
decorative and background objects, internet object family, archive-native furniture, legacy web ecology, social-interface debris, digital folk relics

## Taxonomy Seeds
1. build a composition only from decorative and background objects and let the subject emerge from the clutter
2. use decorative and background objects as floating iconographic fragments around a portrait or shrine page
3. arrange decorative and background objects into a dense flat-lay of internet life-support machinery
4. depict a dead or abandoned page where decorative and background objects remain as the surviving bones
5. treat decorative and background objects like archaeological finds from an extinct social platform
