# Navigation And Link Objects

## Taxonomy Scope
This file groups navigation and link objects within the broader internet-folk style system.

## Main Families
- buttons, nav bars, tabs, breadcrumbs, back/home links, jump lists, blogrolls, side menus, floating anchors
- each family can appear as literal UI furniture, floating collage matter, iconographic relic, or symbolic shorthand

## Era Range
- old web relics
- Y2K and MySpace customization culture
- early social-web residue
- forum and gallery communities
- meme and repost archaeology

## Aesthetic Roles
- functional object: something the page or platform actually uses
- decorative object: something added for personality or clutter
- identity object: something that signals taste, mood, status, or affiliation
- relic object: something that shows age, damage, dead links, or prior circulation

## Prompt Language
navigation and link objects, internet object family, archive-native furniture, legacy web ecology, social-interface debris, digital folk relics

## Taxonomy Seeds
1. build a composition only from navigation and link objects and let the subject emerge from the clutter
2. use navigation and link objects as floating iconographic fragments around a portrait or shrine page
3. arrange navigation and link objects into a dense flat-lay of internet life-support machinery
4. depict a dead or abandoned page where navigation and link objects remain as the surviving bones
5. treat navigation and link objects like archaeological finds from an extinct social platform
