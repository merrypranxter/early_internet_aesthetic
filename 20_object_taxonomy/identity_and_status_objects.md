# Identity And Status Objects

## Taxonomy Scope
This file groups identity and status objects within the broader internet-folk style system.

## Main Families
- avatars, userpics, rank bars, mood icons, away messages, status dots, usernames, signatures, friend badges
- each family can appear as literal UI furniture, floating collage matter, iconographic relic, or symbolic shorthand

## Era Range
- old web relics
- Y2K and MySpace customization culture
- early social-web residue
- forum and gallery communities
- meme and repost archaeology

## Aesthetic Roles
- functional object: something the page or platform actually uses
- decorative object: something added for personality or clutter
- identity object: something that signals taste, mood, status, or affiliation
- relic object: something that shows age, damage, dead links, or prior circulation

## Prompt Language
identity and status objects, internet object family, archive-native furniture, legacy web ecology, social-interface debris, digital folk relics

## Taxonomy Seeds
1. build a composition only from identity and status objects and let the subject emerge from the clutter
2. use identity and status objects as floating iconographic fragments around a portrait or shrine page
3. arrange identity and status objects into a dense flat-lay of internet life-support machinery
4. depict a dead or abandoned page where identity and status objects remain as the surviving bones
5. treat identity and status objects like archaeological finds from an extinct social platform
