# NEGATIVES

Use these as guardrails when the output starts getting too polished, too modern, or too tasteful.

## Global negatives
- pristine flat-design UI
- tasteful editorial minimalism
- clean luxury branding
- modern startup mockup vibes
- cinematic naturalism with no interface clutter
- perfect anti-aliased vector poster cleanliness
- sterile monochrome systems
- brand-safe neutrality
- “elegant” muted restraint
- over-smoothed photoreal surfaces
- silent empty composition with no social trace

## Old web negatives
- responsive modern layouts
- contemporary web-app polish
- high-end product photography
- invisible code / invisible structure

## MySpace negatives
- restrained layouts
- neutral palettes
- adult-professional self-presentation
- sophisticated social UI

## Meme negatives
- humorless sincerity
- over-explained design
- perfect text alignment
- clean message hierarchy

## Glitch negatives
- subtle film grain instead of actual artifact logic
- balanced tasteful chromatic aberration
- cyberpunk city defaults
- sleek futuristic cleanliness

## Folk-internet negatives
- single-author preciousness
- clean museum presentation
- no evidence of reposting or re-use
- no archive scars
