# Neopets Browser Magic Megafile

## Core Identity
Neopets Browser Magic Megafile is treated here as a full stylistic ecosystem rather than a couple of nostalgic references. The goal is to preserve the platform's specific emotional weather, decorative habits, interface logic, and social performance rituals.

## Core Vibe
flash-browser wonder, neopoints labor, fantasy UI cuteness, childish web sorcery

## Visual Markers
- platform-specific UI fragments, buttons, comment boxes, status widgets, or profile panels
- low-res screenshots, repeated crops, repost damage, and visible generational loss
- identity performance through decoration, icons, screen names, badges, stamps, and quote text
- emotionally loaded typography: tiny script, crunchy outlines, glow text, lyric captions, or deadpan labels
- browser-native clutter instead of clean poster composition

## Emotional Tone
This branch should keep the original internet feeling intact: too much feeling, too much decoration, too much irony, too much attachment to tiny meaningless objects that somehow carry a whole personality.

## Composition Behavior
- layered screenshots over independent decorative junk
- asymmetry, collage accumulation, and edge clutter
- profile-like framing, banner-like compression, or dashboard-like vertical drift
- loops of reposting, quoting, reblogging, bumping, or revising

## Prompt Language
flash-browser wonder, neopoints labor, fantasy UI cuteness, childish web sorcery, browser-window residue, timestamp fossils, cursor ghosts, emotional overposting, profile-performance logic, repost palimpsest, hand-coded or faux-hand-coded page energy
