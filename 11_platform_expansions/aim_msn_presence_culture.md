# AIM/MSN Presence Culture

## Definition
A platform-specific branch of internet folk aesthetics built around buddy lists, away messages, nudge drama, emoticon flirtation, status-line intimacy.

## Core Vibe
buddy lists, away messages, nudge drama, emoticon flirtation, status-line intimacy

## Visual Traits
- buddy icons
- away message poetry
- status glow
- notification sounds
- chat windows
- pixel hearts

## Materials / Texture
- screenshot layering
- low-res repost logic
- interface residue treated as ornament
- user-authored clutter instead of clean composition

## Social Logic
This branch matters because the platform's behavioral rules become the look: dashboards create collage,
buddy lists create intimacy, forums create performative identity, item economies create status glamour, and
anonymous boards create swarm-style authorship.

## Prompt Language
buddy icons, away message poetry, status glow, notification sounds, chat windows, pixel hearts, browser-frame residue, repost wear, emotional self-curation, digital folk texture

## Pairings
Works well with glitch rot, scene/emo, vaporwave corporate afterlife, weirdcore, or sacred absurdity depending on tone.

## Negatives
Sterile corporate UI, tasteful minimalism, contemporary app flatness, polished cinematic realism, generic modern influencer content.
