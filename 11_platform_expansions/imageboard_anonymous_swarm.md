# Imageboard Anonymous Swarm

## Definition
A platform-specific branch of internet folk aesthetics built around green text, blunt cruelty, anonymity, inside jokes, low-fi screenshot folkways.

## Core Vibe
green text, blunt cruelty, anonymity, inside jokes, low-fi screenshot folkways

## Visual Traits
- greentext formatting
- anonymous posting
- reaction images
- thread drift
- ironic detachment
- swarm authorship

## Materials / Texture
- screenshot layering
- low-res repost logic
- interface residue treated as ornament
- user-authored clutter instead of clean composition

## Social Logic
This branch matters because the platform's behavioral rules become the look: dashboards create collage,
buddy lists create intimacy, forums create performative identity, item economies create status glamour, and
anonymous boards create swarm-style authorship.

## Prompt Language
greentext formatting, anonymous posting, reaction images, thread drift, ironic detachment, swarm authorship, browser-frame residue, repost wear, emotional self-curation, digital folk texture

## Pairings
Works well with glitch rot, scene/emo, vaporwave corporate afterlife, weirdcore, or sacred absurdity depending on tone.

## Negatives
Sterile corporate UI, tasteful minimalism, contemporary app flatness, polished cinematic realism, generic modern influencer content.
