# LiveJournal Overshare

## Definition
A platform-specific branch of internet folk aesthetics built around coded intimacy, diary confession, fandom icons, mood themes, html skin sadness.

## Core Vibe
coded intimacy, diary confession, fandom icons, mood themes, html skin sadness

## Visual Traits
- 100x100 icons
- friends-only aura
- mood themes
- comment intimacy
- fandom signatures
- confessional text blocks

## Materials / Texture
- screenshot layering
- low-res repost logic
- interface residue treated as ornament
- user-authored clutter instead of clean composition

## Social Logic
This branch matters because the platform's behavioral rules become the look: dashboards create collage,
buddy lists create intimacy, forums create performative identity, item economies create status glamour, and
anonymous boards create swarm-style authorship.

## Prompt Language
100x100 icons, friends-only aura, mood themes, comment intimacy, fandom signatures, confessional text blocks, browser-frame residue, repost wear, emotional self-curation, digital folk texture

## Pairings
Works well with glitch rot, scene/emo, vaporwave corporate afterlife, weirdcore, or sacred absurdity depending on tone.

## Negatives
Sterile corporate UI, tasteful minimalism, contemporary app flatness, polished cinematic realism, generic modern influencer content.
