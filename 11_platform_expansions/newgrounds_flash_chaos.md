# Newgrounds Flash Chaos

## Definition
A platform-specific branch of internet folk aesthetics built around vector flash violence, edgy humor, thick outlines, portal culture, button-heavy cartoon energy.

## Core Vibe
vector flash violence, edgy humor, thick outlines, portal culture, button-heavy cartoon energy

## Visual Traits
- Flash tweening
- portal UI
- thick vector line
- edgy mascot faces
- loading screen bravado
- game menu red-black palette

## Materials / Texture
- screenshot layering
- low-res repost logic
- interface residue treated as ornament
- user-authored clutter instead of clean composition

## Social Logic
This branch matters because the platform's behavioral rules become the look: dashboards create collage,
buddy lists create intimacy, forums create performative identity, item economies create status glamour, and
anonymous boards create swarm-style authorship.

## Prompt Language
Flash tweening, portal UI, thick vector line, edgy mascot faces, loading screen bravado, game menu red-black palette, browser-frame residue, repost wear, emotional self-curation, digital folk texture

## Pairings
Works well with glitch rot, scene/emo, vaporwave corporate afterlife, weirdcore, or sacred absurdity depending on tone.

## Negatives
Sterile corporate UI, tasteful minimalism, contemporary app flatness, polished cinematic realism, generic modern influencer content.

visible interface history, decorative clutter, emotionally loaded customization rather than professional design.
