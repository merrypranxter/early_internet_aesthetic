# Tumblr 2010s

## Definition
A platform-specific branch of internet folk aesthetics built around pastel overshare + fandom collage + poetic irony + png witchcraft.

## Core Vibe
pastel overshare + fandom collage + poetic irony + png witchcraft

## Visual Traits
- transparent PNG cutouts
- italic lowercase captions
- fandom shrines
- reaction gifs
- soft blur melancholy
- dashboard collage

## Materials / Texture
- screenshot layering
- low-res repost logic
- interface residue treated as ornament
- user-authored clutter instead of clean composition

## Social Logic
This branch matters because the platform's behavioral rules become the look: dashboards create collage,
buddy lists create intimacy, forums create performative identity, item economies create status glamour, and
anonymous boards create swarm-style authorship.

## Prompt Language
transparent PNG cutouts, italic lowercase captions, fandom shrines, reaction gifs, soft blur melancholy, dashboard collage, browser-frame residue, repost wear, emotional self-curation, digital folk texture

## Pairings
Works well with glitch rot, scene/emo, vaporwave corporate afterlife, weirdcore, or sacred absurdity depending on tone.

## Negatives
Sterile corporate UI, tasteful minimalism, contemporary app flatness, polished cinematic realism, generic modern influencer content.

A scene rendered through **tumblr 2010s** logic: transparent PNG cutouts, italic lowercase captions, fandom shrines, reaction gifs, repost grain, browser-window detritus,
visible interface history, decorative clutter, emotionally loaded customization rather than professional design.
