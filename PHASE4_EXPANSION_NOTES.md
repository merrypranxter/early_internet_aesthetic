# Phase 4 Expansion Notes

Phase 4 shifts the repo from broad taxonomy into flagship deployment material.

## Major additions
- 8 flagship docs in `12_flagship_docs/`
- each flagship doc contains a core identity section, hero prompts, and 100 polished seeds
- 18 site-specific object docs in `13_site_specific_objects/`
- each object doc contains variants and 40 object-focused prompt seeds
- new `hero_prompts_flagship.md`
- appendix updates for phase tracking and object mapping

## Intent
This phase makes the repo easier to use directly for image prompting, moodboarding, and future agent ingestion. The flagship docs function like style anchors. The site-specific object docs make platform scenes feel materially accurate instead of vaguely retro.
