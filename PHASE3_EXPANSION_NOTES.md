# Phase 3 Expansion Notes

Phase 3 pushes the repo from broad taxonomy into deeper platform-native style engines.

## What this phase adds
- platform-specific mega files with large prompt-seed banks
- dedicated sub-branches for scene/emo web, Tumblr sadgirl/weirdgirl, DeviantArt fantasy kitsch, and forum signature/banner culture
- expanded prompt modules for avatar portraits, profile pages, sig banners, and relic objects
- larger reference lists for usernames, decorative UI junk, and scene-era emotional language
- merge recipes for platform-native chaos crossed with dreamcore, glam, sacred absurdity, and print grime

## Intended use
These files are built to be copied directly into prompts, mined for tags, remixed into agent instructions, or used as stylistic training notes for future repos.
