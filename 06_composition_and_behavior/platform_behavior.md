# Platform Behavior

## Definition
Composition rules generated by platform affordances rather than traditional art-school principles.

## Key behaviors
- dashboard stacking and reblog nesting
- sidebars, modules, and profile boxes as framing devices
- comments and signatures behaving like decorative borders
- status messages functioning as emotional subtitles
- inventory panes and avatar items creating dense ornament
- thread drift and reply branching replacing linear narrative

## Prompt Language
dashboard nesting, profile-box framing, signature-border clutter, status-line subtitle, thread branching, sidebar ornament, module stacking
