# Tumblr Sadgirl Weirdgirl

## Definition
Tumblr sadgirl/weirdgirl is a mood-board identity engine built from diary fragments, gifsets, crying glitter, velvet darkness, literary quotations, ironic softness, and a haunted-bedroom version of internet selfhood.

## Core Vibe
Overdesigned, emotionally legible, platform-native, and visibly handmade. This branch favors strong identity performance through decorative compression rather than tasteful restraint.

## Visual Traits
- glow-heavy text and clipped gradients
- layered icons, badges, and tiny image fragments
- visible low-res compression and screenshot wear
- decorative framing devices, borders, dividers, and stamps
- text treated as emotional object rather than neutral information

## Materials / Texture
- JPEG crust, glitter overlays, CRT-ish bloom, webcam flash, faux metallic brushes, transparent PNG junk

## Common Motifs
- profile photos, signature banners, quote text, hearts/skulls/stars, avatars, stamps, dividers, comment panels

## Palette
High-contrast black + neon, lavender dusk, icy blue chrome, sickly pink glow, or fantasy moonlight bloom depending on subject.

## Composition
Compression-friendly crops, horizontal signature ratios, dashboard stacks, profile-page arrangements, or collage walls.

## Prompt Language
screen-name glamour, browser residue, dramatic captioning, low-res emotional theater, platform-native decoration, stamp clutter, quote-graphic energy

## Pairings
Works well with glitchcore, scrapbook maximalism, vaporwave, weirdcore, camp, printshop grunge, and cyber bubblegum.

## Negatives
- tasteful editorial design
- minimalist typography
- frictionless modern interfaces
- neutral photography with no social residue
