# Internet Folk X Cyber Bubblegum

## Merge Thesis
This merge combines internet-folk vernacular with holographic chat windows, candy-chrome UI ruins, bubble icons, pink rave desktop glow, flirtatious machine sweetness.

## Keep From Internet Folk
- visible repost lineage
- screenshot / UI residue
- collective, handmade, or customized mess
- irony and sincerity coexisting in the same frame

## Add From Secondary Style
- holographic chat windows, candy-chrome UI ruins, bubble icons, pink rave desktop glow, flirtatious machine sweetness

## Watch Outs
Do not let the secondary style sterilize the vernacular weirdness. Keep browser debris, cheap texture, or user-authored clutter alive.

## Prompt Formula
subject + internet-folk clutter + holographic chat windows, candy-chrome UI ruins, bubble icons, pink rave desktop glow, flirtatious machine sweetness + platform residue + visible emotional customization + anti-corporate mess-maximalism
