# Internet Folk X Kawaii

## Merge Thesis
This merge combines internet-folk vernacular with cute sticker overload, pastel browser relics, avatar sweetness, toybox UI, glitter affection, soft plastic optimism.

## Keep From Internet Folk
- visible repost lineage
- screenshot / UI residue
- collective, handmade, or customized mess
- irony and sincerity coexisting in the same frame

## Add From Secondary Style
- cute sticker overload, pastel browser relics, avatar sweetness, toybox UI, glitter affection, soft plastic optimism

## Watch Outs
Do not let the secondary style sterilize the vernacular weirdness. Keep browser debris, cheap texture, or user-authored clutter alive.

## Prompt Formula
subject + internet-folk clutter + cute sticker overload, pastel browser relics, avatar sweetness, toybox UI, glitter affection, soft plastic optimism + platform residue + visible emotional customization + anti-corporate mess-maximalism

## Example Snippet
A composition full of cute sticker overload, pastel browser relics, avatar sweetness, toybox UI, glitter affection, soft plastic optimism, but still scarred by repost life: window frames, cropped screenshots, low-res remnants,
status-line fragments, and anonymous folk texture.
