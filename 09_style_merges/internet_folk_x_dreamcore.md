# Internet Folk X Dreamcore

## Merge Logic
Dreamcore adds liminal hallways, blurred nostalgia, hush, and eerie emotional suspension to browser-native clutter.

## Keep From Internet Folk
- screenshot strata
- repost damage
- interface residues
- profile-performance logic
- low-res emotional text behavior

## Add From Crossover
- distinct palette and object family
- emotional or mythic exaggeration
- movement logic or atmosphere unique to the crossover style

## Prompt Formula
internet folk base + platform-native clutter + crossover palette + crossover object family + one social residue + one artifact layer + one emotional caption behavior

## Negatives
Avoid washing the merge into generic cyberpunk, sleek futurism, clean editorial, or abstract mood with no recognizable browser-native material.
