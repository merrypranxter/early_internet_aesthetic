# Internet Folk X Printshop Grunge

## Merge Thesis
This merge combines internet-folk vernacular with halftone failure, cheap flyers, xerox poster clutter, sticker layering, punk web residue, ad-rip collage.

## Keep From Internet Folk
- visible repost lineage
- screenshot / UI residue
- collective, handmade, or customized mess
- irony and sincerity coexisting in the same frame

## Add From Secondary Style
- halftone failure, cheap flyers, xerox poster clutter, sticker layering, punk web residue, ad-rip collage

## Watch Outs
Do not let the secondary style sterilize the vernacular weirdness. Keep browser debris, cheap texture, or user-authored clutter alive.

## Prompt Formula
subject + internet-folk clutter + halftone failure, cheap flyers, xerox poster clutter, sticker layering, punk web residue, ad-rip collage + platform residue + visible emotional customization + anti-corporate mess-maximalism

## Example Snippet
A composition full of halftone failure, cheap flyers, xerox poster clutter, sticker layering, punk web residue, ad-rip collage, but still scarred by repost life: window frames, cropped screenshots, low-res remnants,
status-line fragments, and anonymous folk texture.
